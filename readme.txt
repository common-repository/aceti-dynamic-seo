Dynamically generates Search Engine Optimizing code for your Wordpress powered site.

== Description ==

Aceti will optimize your site for search engine indexing.

Aceti Dynamic SEO offers:

* Optimizes titles for Search Engines.
* Generates meta tags automatically.
* Override controls so you can  set meta tags manually on a case-by-case basis.
* Custom post type support.
* Aceti is the highest performing SEO plugin; uses 70% less resources than the highest performing SEO plugin. 90% less than the most popular SEO plugin. Loads fewer than 390 lines of code for non-admin page views.
* Multiple layers of security and i18n-ready.
* Canonical and index tags help control duplicate content.
* Choose between hook-only or output buffering: It's up to you.

Visit [Aceti Dynamic SEO's Page](http://citrinedesign.net/blog/projects/aceti) for more information and support.

== Installation ==

Use WordPress plugin management tools to automatically install Aceti, or download a copy and upload manually:

1. Upload `aceti-dynamic-seo.zip` to your `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Follow simple set up instructions.
1. All done!
